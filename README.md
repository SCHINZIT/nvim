# Neovim config dir


## Prequesites

**deoplete** && **ternjs**  

Python must be enabled. So install with pip2 / pip3.  

`pip2 install neovim`  
`pip3 install neovim`  

Node.js and npm must be installed in order to run ternjs.  

## Install Addons/Plugins

Start neovim and call `:PlugInstall`.
